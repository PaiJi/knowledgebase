## 邮件 SAAS

### [Loops](https://loops.so/)

支持以编程形式来发送邮件的产品，免费套餐较为慷慨：

```
Up to 1,000 contacts
Up to 2,000 sends/month
```

### [Resend](https://resend.com/)

和 Loops 差不多定位的产品。免费套餐配额如下：

```
Up to 3,000 emails / mo
100 emails / day
Email Support
1 Domain
1-day Data Retention
```
